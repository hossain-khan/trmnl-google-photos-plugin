# Plugin Icons

Place plugin icons in this directory:

- `icon.png` - Main plugin icon (recommended: 512x512px)
- `icon@2x.png` - Retina version (1024x1024px)

## Design Guidelines

- Use Google Photos brand colors or camera/photo imagery
- Keep it simple and recognizable at small sizes
- Square aspect ratio
- PNG format with transparency
- Related to photography/albums theme
